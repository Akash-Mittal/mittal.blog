How to Get Spider-Verse Web Shooters in Fortnite

Become a web-slinging superhero with this guide.

Remember that scene in the movie Spider-Man: Into the Spider-Verse where Miles Morales discovers his new spider-powers and creates his own web-shooters? Well, now you can do the same in Fortnite.

Epic Games has added a new weapon to the game called the "Venom Blade", which can be transformed into "Spider-Verse Web Shooters" by completing a series of challenges. Once you have the web-shooters equipped, you can swing around the island just like Spider-Man.

Here are the challenges you need to complete to unlock the Spider-Verse Web Shooters:

  1. Deal 500 damage with the Venom Blade
  2. Eliminate 25 opponents with the Venom Blade
  3. Swing 3,000 meters with the Venom Blade equipped
  4. Use the Venom Blade to destroy 50 structures
  5. Use the Venom Blade to harvest 500 resources

Completing these challenges will reward you with the Spider-Verse Web Shooters, which can be found in the "Miscellaneous" section of your inventory.
